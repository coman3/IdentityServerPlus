A bloody massive extension on top of IdentityServer4 that makes it actualy usefull to the lazy programer.
Primary Goals:
 - [x] Plugable
 - [ ] Easy to setup
 - [ ] Scalable and Relabile
 - [ ] Secure
 - [ ] Faster than the crap that exsists today
